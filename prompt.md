Технический стек: Framework: Next.js 15+ (App Router). Стиль: Tailwind CSS 4.0+. Компоненты: Radix UI или Shadcn UI для доступности. Иконки: Lucide-React. Локализация: next-intl или кастомное решение на словарях (RU, KZ, EN). 1. Дизайн и Эстетика (Дорогой минимализм): Цветовая палитра: Светлая тема: Основной фон — мягкий бежевый (#F5F5DC или bg-stone-50), акценты — глубокий оранжевый (#FF8C00), текст — антрацитовый. Тёмная тема: Основной фон — глубокий графитовый (не черный, bg-[#1a1a1a]), акценты — яркий оранжевый, дополнительные оттенки — серо-коричневый. Стиль: Скругленные углы (xl/2xl), тонкие границы (border-stone-200/800), эффект размытия (backdrop-blur) для модальных окон, плавные микро-анимации через Framer Motion. 2. SEO и Мультиязычность: Архитектура URL: /[lang]/[category]/[tool-name] (например, /ru/text/word-counter). SEO-компоненты: * Автоматическая генерация alternate ссылок (hreflang) для RU, KZ, EN. Динамические title и description для каждой из 100 страниц на 3 языках. Микроразметка JSON-LD SoftwareApplication для каждой утилиты. Блок "FAQ" и "Инструкция" внизу каждой страницы для индексации текстов. 3. Безопасность и Скорость: Все вычисления — строго на клиенте ("use client"). Использование DOMPurify для очистки вводимых данных. Минификация кода и обфускация критической логики при сборке. Максимальный балл Core Web Vitals (LCP < 1.2s). 4. Структура 100 утилит (по категориям): Текст: Счетчики, конвертеры регистра, транслит, шрифты для соцсетей, очистка пробелов, Zalgo, генератор Lorem Ipsum. Рандом: Колесо фортуны, кубики, генератор паролей, UUID, перемешивание списков. Время: Помодоро, секундомер, мировое время, таймеры, Unix-конвертер. Математика: % калькулятор, ИМТ, кредит, НДС, системы счисления. Графика: Сжатие (canvas), Color Picker, CSS градиенты, Base64, Favicon gen. Dev-tools: JSON Formatter, JWT Decoder, HTML/URL Encode, RegEx тестер. Соцсети: UTM-метки, теги YT, лимиты символов, генератор ссылок WA. Конвертеры: Вес, длина, валюты (статичный курс), римские цифры. Крипто: MD5, SHA-256, AES (crypto-js), азбука Морзе. Разное: Тест кликов (CPS), метроном, блокнот (localStorage), линейка. Инструкция для выполнения: Инициализируй проект Next.js с поддержкой переключения тем (next-themes). Создай Header с переключателем языков (RU/KZ/EN) и темы.
